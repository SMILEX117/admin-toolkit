# Admin-Toolkit

Eine Sammlung nützlicher Bash-Skripte für die tägliche Systemadministration.

## Enthaltene Skripte
* `backup.sh`: Erstellt komprimierte Backups von wichtigen Verzeichnissen.
* `usercheck.sh`: Zeigt aktuell angemeldete Benutzer an.
* `diskspace.sh`: Überprüft die Festplattenbelegung.
* `systeminfo.sh`: Gibt Kerninformationen zum System (Kernel, Hostname) aus.
